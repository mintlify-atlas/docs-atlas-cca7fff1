## Herramientas del curso

<Steps>
  <Step title="PostgreSQL / Supabase">
    El motor principal del curso. Usaremos [Supabase](https://supabase.com) como plataforma cloud de PostgreSQL para ejecutar todos los scripts SQL del curso.
  </Step>
  <Step title="RelaX">
    Herramienta web para evaluar expresiones de álgebra relacional de forma interactiva. Disponible en [dbis-uibk.github.io/relax](https://dbis-uibk.github.io/relax/).
  </Step>
  <Step title="Google Colab">
    Entorno de notebooks Jupyter en la nube para el taller de pgvector. No requiere instalación local.
  </Step>
  <Step title="draw.io">
    Herramienta de diagramación para crear modelos E-R y EER. Disponible en [app.diagrams.net](https://app.diagrams.net/).
  </Step>
</Steps>

<Note>
  Todo el material del curso está disponible en el repositorio GitHub. Los archivos `.ipynb` pueden abrirse directamente en Google Colab y los scripts `.sql` se ejecutan en el editor SQL de Supabase.
</Note>
